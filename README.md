# Сайт "Центр подготовки специалистов Амур"

## Структура сайта

### Основные страницы:
1. **index.html** - Главная страница с фоном и видео-превью
2. **history.html** - История подразделения
3. **charter.html** - Устав ВС СССР (макет как на последнем скрине)

### Методичка (в папке methodics/):
- **general.html** - Общая документация
- **sergeant-school.html** - Сержантская школа
- **sanitary.html** - Санитарная книга
- **charter.html** - Уставы

Все страницы методички имеют такой же макет как charter.html (с боковой навигацией).

## Как добавить контент

### Для добавления текста на обычных страницах (index, history):
Измените секцию:
```html
<div class="text-content">
    <!-- Ваш текст здесь -->
</div>
```

### Для добавления контента на страницах методички/устава:
Измените секцию:
```html
<div class="charter-text-content">
    <div class="intro-text">
        <p>Ваш текст здесь</p>
    </div>
</div>
```

### HTML теги для оформления текста:

**Заголовки:**
```html
<h2 class="section-title">ЗАГОЛОВОК</h2>
```

**Списки:**
```html
<ul class="chapters-list">
    <li><a href="#">Пункт 1</a></li>
    <li><a href="#">Пункт 2</a></li>
</ul>
```

**Выделение текста:**
```html
<span class="highlight">Важный текст</span>
```

**Разделитель:**
```html
<div class="text-divider"></div>
```

**Абзацы:**
```html
<p>Обычный текст</p>
```

## Запуск

Просто откройте index.html в браузере. Все стили подключены через styles.css.

## Требования

Не требует сервера, работает как статический HTML сайт.
